# homebrew-claude-insights

Homebrew tap for [claude-insights](https://github.com/infowhere-ai/claude-insights) — real-time dashboard for Claude Code sessions.

## Install

```bash
brew tap infowhere-ai/claude-insights
brew install claude-insights
```

After installing, run:

```bash
claude-insights install  # set up the Claude Code hook
claude-insights start    # open the dashboard at http://localhost:4000
```
